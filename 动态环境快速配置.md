# 动态环境系统快速配置指南

## 快速开始（3步）

### 1. 选择 EnvironmentManager
在 Hierarchy 中选择 `EnvironmentManager` 对象

### 2. 启用动态变化
在 Inspector 中勾选：
```
✓ Enable Dynamic Change
```

### 3. 运行游戏
点击 Play 按钮，环境开始自动变化！

## 推荐配置

### 标准游戏模式
```
Enable Dynamic Change: ✓
Temperature Change Speed: 3
Humidity Change Speed: 4
Direction Change Interval: 3
Temperature Range: (10, 40)
Humidity Range: (30, 90)
```

**效果**：
- 温度每秒变化约3度
- 湿度每秒变化约4%
- 每3秒可能改变方向
- 需要玩家持续关注和调节

## 游戏玩法

### 目标
在30秒内让橘子树完成生长，同时保持环境适宜

### 操作
- **红色圆形按钮（上三角）**：增加温度
- **蓝色圆形按钮（下三角）**：降低温度
- **红色圆形按钮（上三角）**：增加湿度
- **蓝色圆形按钮（下三角）**：降低湿度

### 适宜范围
- **温度**: 15°C - 35°C（绿色安全区）
- **湿度**: 40% - 80%（绿色安全区）

### 危险提示
- 温度或湿度超出范围 → 树木死亡 💀
- 树木死亡 → 停止生长 ⏸️
- 调节回适宜范围 → 树木恢复 ✅

## 常见问题

**Q: 环境不变化？**  
A: 检查 `Enable Dynamic Change` 是否勾选

**Q: 变化太快？**  
A: 降低 `Change Speed` 参数

**Q: 按钮没反应？**  
A: 检查 EnvironmentController 是否绑定了按钮

**Q: 树一直死？**  
A: 降低变化速度或增加调节步长
