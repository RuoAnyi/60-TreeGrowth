# 产量系统说明

## 功能概述

橘子树生长系统现在包含产量机制：
- 初始产量：50公斤
- 每次环境不适宜导致停止生长时，产量减少1公斤
- 生长达到100%时，显示收获提示

## 核心机制

### 1. 初始产量
```
最大产量: 50kg
初始产量: 50kg
```

### 2. 产量减少规则
**触发条件**：环境从适宜变为不适宜（停止生长）

**判定逻辑**：
```
上一帧：温度和湿度都在适宜范围内 ✅
当前帧：温度或湿度有一个超出范围 ❌
结果：产量 -1kg
```

**示例场景**：
```
场景1：温度从30°C升到38°C
- 30°C: 适宜 ✅
- 38°C: 超出37°C上限 ❌
- 结果: 产量 -1kg

场景2：湿度从70%降到60%
- 70%: 适宜 ✅
- 60%: 低于65%下限 ❌
- 结果: 产量 -1kg

场景3：持续在不适宜范围内
- 第1帧: 40°C (不适宜) ❌
- 第2帧: 41°C (不适宜) ❌
- 结果: 不减少（只在刚停止时减少一次）
```

### 3. 收获提示
**触发条件**：生长值达到100%

**显示内容**：
```
恭喜您！您已收获 [当前产量] 公斤的橘子！
```

**示例**：
- 完美生长（从未停止）：`恭喜您！您已收获 50 公斤的橘子！`
- 停止5次：`恭喜您！您已收获 45 公斤的橘子！`
- 停止20次：`恭喜您！您已收获 30 公斤的橘子！`

## 代码实现

### OrangeTreeController.cs

#### 新增字段
```csharp
[Header("产量系统")]
[SerializeField] private int maxYield = 50;           // 最大产量
[SerializeField] private int currentYield = 50;       // 当前产量
[SerializeField] private int yieldLossPerStop = 1;    // 每次停止损失

private bool wasGrowingSuitableLastFrame = true;     // 上一帧环境状态
private bool hasHarvested = false;                    // 是否已收获
```

#### 新增事件
```csharp
public event Action<int> OnYieldChanged;      // 产量变化事件
public event Action<int> OnHarvestComplete;   // 收获完成事件
```

#### 核心方法
```csharp
// 生长停止时调用
private void OnGrowthStopped()
{
    currentYield = Mathf.Max(0, currentYield - yieldLossPerStop);
    OnYieldChanged?.Invoke(currentYield);
}

// 达到100%时调用
private void OnHarvestReached()
{
    hasHarvested = true;
    OnHarvestComplete?.Invoke(currentYield);
}
```

### HarvestMessageUI.cs
收获消息UI组件，用于显示收获提示。

**功能**：
- 监听 `OnHarvestComplete` 事件
- 显示收获消息面板
- 自动隐藏（可配置）

### YieldDisplayUI.cs
产量显示UI组件，实时显示当前预期产量。

**功能**：
- 监听 `OnYieldChanged` 事件
- 实时更新产量显示
- 根据产量设置颜色（正常/警告/危险）

## UI设置

### 1. 收获消息面板
需要在场景中创建：
```
Canvas
└── HarvestMessagePanel (GameObject)
    ├── Background (Image)
    ├── MessageText (Text) - "恭喜您！您已收获 XX 公斤的橘子！"
    └── CloseButton (Button)
```

添加 `HarvestMessageUI` 组件到 `HarvestMessagePanel`，并连接引用。

### 2. 产量显示
需要在场景中创建：
```
Canvas
└── YieldDisplay (GameObject)
    └── YieldText (Text) - "预期产量: 50kg"
```

添加 `YieldDisplayUI` 组件到 `YieldDisplay`，并连接引用。

## 游戏体验

### 完美通关（50kg）
```
1. 点击"开始生长"
2. 持续调节温度和湿度，保持在适宜范围内
3. 30秒后达到100%
4. 显示："恭喜您！您已收获 50 公斤的橘子！"
```

### 普通通关（30-49kg）
```
1. 点击"开始生长"
2. 偶尔环境超出范围，停止生长几次
3. 产量减少到30-49kg
4. 最终达到100%
5. 显示："恭喜您！您已收获 XX 公斤的橘子！"
```

### 困难通关（1-29kg）
```
1. 点击"开始生长"
2. 频繁环境超出范围，多次停止生长
3. 产量大幅减少
4. 最终达到100%
5. 显示："恭喜您！您已收获 XX 公斤的橘子！"
```

### 最低产量（0kg）
```
1. 停止生长超过50次
2. 产量降到0kg
3. 达到100%时显示："恭喜您！您已收获 0 公斤的橘子！"
```

## 难度平衡

### 当前设置
- 生长时间：30秒
- 初始产量：50kg
- 每次停止损失：1kg
- 允许停止次数：50次

### 难度分析
```
完美通关：0次停止 → 50kg
优秀通关：1-10次停止 → 40-49kg
良好通关：11-20次停止 → 30-39kg
及格通关：21-30次停止 → 20-29kg
勉强通关：31-49次停止 → 1-19kg
失败通关：50次以上停止 → 0kg
```

### 调整建议

**如果太难**：
```csharp
yieldLossPerStop = 0.5f;  // 每次损失0.5kg
maxYield = 100;           // 提高最大产量
```

**如果太简单**：
```csharp
yieldLossPerStop = 2;     // 每次损失2kg
maxYield = 30;            // 降低最大产量
```
