# 🎨 最简单的纹理导入方法

## 不需要 "Embed Textures"！

### 第1步：找到纹理文件

#### 在 Blender 中查找纹理位置：

1. **切换到 Shading 工作区**
   - 顶部标签：Shading

2. **选择你的模型**
   - 在 3D 视图中点击橘子树

3. **查看材质节点**
   - 在下方的 Shader Editor 中
   - 找到 Image Texture 节点（橙色的）

4. **查看文件路径**
   - 点击 Image Texture 节点
   - 鼠标悬停在图片名称上
   - 会显示完整路径

5. **记下路径**
   - 例如：`C:/Users/你的用户名/Documents/Blender/textures/`

#### 或者使用文件菜单：

1. **File** → **External Data** → **Report Missing Files**
2. 查看控制台输出的文件路径

---

### 第2步：复制纹理文件到 Unity

#### 找到这些文件：

根据你的截图，你的纹理文件名是：
```
texture_pbr_20250901.png          (颜色贴图)
texture_pbr_20250901_normal.png   (法线贴图)
texture_pbr_20250901_roughness.png (粗糙度)
texture_pbr_20250901_metallic.png  (金属度)
```

#### 复制步骤：

1. **打开文件资源管理器**
   - 找到 Blender 项目文件夹
   - 或者根据第1步找到的路径

2. **找到纹理文件**
   - 找到上面列出的4个PNG文件

3. **复制文件**
   - 选中所有纹理文件
   - Ctrl+C 复制

4. **粘贴到 Unity 项目**
   - 打开：`TreePlanQAQ/Assets/Textures/`
   - 如果没有这个文件夹，创建一个
   - Ctrl+V 粘贴

---

### 第3步：导出 FBX（简单设置）

在 Blender 中：

1. **选择模型**
   - 点击橘子树

2. **导出 FBX**
   - File → Export → FBX (.fbx)

3. **基本设置**（就用你截图中的设置）：
   ```
   Path Mode: Copy
   Include:
   ✅ Limit to: Selected Objects
   Object Types: Mesh
   ```

4. **保存位置**
   - 保存到：`TreePlanQAQ/Assets/Models/OrangeTree.fbx`

5. **点击 Export FBX**

---

### 第4步：在 Unity 中设置材质

#### 4.1 设置法线贴图

1. **在 Unity Project 窗口**
   - 找到 `Assets/Textures/texture_pbr_20250901_normal.png`

2. **选中它**
   - 在 Inspector 中

3. **修改设置**
   ```
   Texture Type: Normal map  ← 改为这个
   ```

4. **点击 Apply**

#### 4.2 创建材质

1. **在 Project 窗口右键**
   - Create → Material

2. **命名**
   - `OrangeTreeMaterial`

3. **设置材质**
   - 选中材质
   - 在 Inspector 中：

   ```
   Shader: Standard (或 Universal Render Pipeline/Lit)
   
   Albedo:
   - 点击小方框
   - 选择 texture_pbr_20250901.png
   
   Normal Map:
   - 点击小方框
   - 选择 texture_pbr_20250901_normal.png
   
   Metallic:
   - 如果有金属贴图，拖入
   - 或者设置为 0
   
   Smoothness:
   - 调整滑块（0.5 左右）
   - 或者使用粗糙度贴图
   ```

#### 4.3 应用材质到模型

1. **在 Hierarchy 中找到橘子树**
   - 展开 OrangeTree.fbx

2. **拖拽材质**
   - 将 OrangeTreeMaterial 拖到模型上

3. **完成！**
   - 现在应该能看到颜色了

## ❓ 如果找不到纹理文件

### 方法1：在 Blender 中打包并提取

1. **打包资源**
   - File → External Data → Pack Resources
   - 所有纹理会被打包到 .blend 文件中

2. **解包资源**
   - File → External Data → Unpack Resources
   - 选择 "Write files to current directory"
   - 纹理会被提取到 .blend 文件所在文件夹

3. **复制到 Unity**
   - 找到提取的纹理文件
   - 复制到 Unity 项目

### 方法2：重新保存纹理

1. **在 Shader Editor 中**
   - 选择 Image Texture 节点

2. **保存图片**
   - Image → Save As
   - 保存到一个你知道的位置

3. **复制到 Unity**
