# 生长速度计算说明

## 当前生长计算公式

```csharp
生长速度 = baseGrowthRate × envFactor × Time.deltaTime
```

其中：
- `baseGrowthRate`：基础生长速率（当前为 2.4，已调快两倍）
- `envFactor`：环境因子（0-1之间）
- `Time.deltaTime`：帧时间间隔

## 环境因子计算

环境因子由三个部分组成：

```csharp
envFactor = tempFactor × 0.4 + humidFactor × 0.3 + sunFactor × 0.3
```

### 各因子权重
- 温度权重：**40%**
- 湿度权重：**30%**
- 光照权重：**30%**

## 默认环境参数

### OrangeTreeController 默认值
```csharp
temperature = 25°C
humidity = 60%
sunlight = 600 lux
```

### GrowthConfig 最佳值（代码中）
```csharp
optimalTemperature = 26°C (容差 ±13.2°C)
optimalHumidity = 75% (容差 ±7.5%)
optimalSunlight = 600 lux (容差 ±300 lux)
```

## 环境因子计算示例

### 使用默认值 (25°C, 60%, 600lux)

#### 1. 温度因子
```
偏差 = |25 - 26| = 1°C
温度因子 = 1 - (1 / 13.2) = 0.924
```

#### 2. 湿度因子
```
偏差 = |60 - 75| = 15%
湿度因子 = 1 - (15 / 7.5) = -1 → 0 (超出容差)
```
⚠️ **问题：** 默认湿度60%偏离最佳值75%达到15%，超过容差7.5%！

#### 3. 光照因子
```
偏差 = |600 - 600| = 0
光照因子 = 1 - (0 / 300) = 1.0
```

#### 总环境因子
```
envFactor = 0.924 × 0.4 + 0 × 0.3 + 1.0 × 0.3
         = 0.3696 + 0 + 0.3
         = 0.6696
```

### 使用最佳值 (26°C, 75%, 600lux)

```
温度因子 = 1.0
湿度因子 = 1.0
光照因子 = 1.0
envFactor = 1.0 × 0.4 + 1.0 × 0.3 + 1.0 × 0.3 = 1.0
```

## 实际生长速度对比

### 默认环境 (25°C, 60%, 600lux)
```
生长速度 = 2.4 × 0.6696 = 1.607 %/秒
完成时间 = 100 / 1.607 ≈ 62秒
```

### 最佳环境 (26°C, 75%, 600lux)
```
生长速度 = 2.4 × 1.0 = 2.4 %/秒
完成时间 = 100 / 2.4 ≈ 42秒
```

## 环境适宜性判断

代码中的 `IsEnvironmentSuitable()` 方法：

```csharp
// 温度范围：26 ± 13.2 = 12.8°C ~ 39.2°C
bool tempOk = temperature >= 12.8 && temperature <= 39.2;

// 湿度范围：75 ± 7.5 = 67.5% ~ 82.5%
bool humidOk = humidity >= 67.5 && humidity <= 82.5;

return tempOk && humidOk;
```

⚠️ **重要发现：**
- 默认湿度 **60%** 不在适宜范围 (67.5% ~ 82.5%) 内
- 这意味着使用默认值时，**生长会停止**！
- 每次停止会减少1kg产量

## 建议调整

### 方案1：调整默认湿度
将 OrangeTreeController 中的默认湿度改为 75%：

```csharp
private float humidity = 75f; // 从60改为75
```

### 方案2：调整湿度容差
将 GrowthConfig 中的湿度容差改大：

```csharp
public float humidityTolerance = 15f; // 从7.5改为15
// 这样范围变为 60% ~ 90%
```

### 方案3：调整最佳湿度
将 GrowthConfig 中的最佳湿度改为 65%：

```csharp
public float optimalHumidity = 65f; // 从75改为65
// 配合容差7.5，范围变为 57.5% ~ 72.5%
```
