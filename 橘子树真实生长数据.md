# 橘子树真实生长数据

## 温度要求

### 最适生长温度
- **范围**: 23～29℃
- **配置**: optimalTemperature = 26°C（中间值）

### 适宜生长温度
- **范围**: 12.8～37℃
- **配置**: temperatureTolerance = 13.2°C
- **计算**: 26 ± 13.2 = 12.8～39.2℃（接近实际范围）

### 生长逻辑
- ✅ **12.8°C ≤ 温度 ≤ 37°C** → 可以生长
- ❌ **温度 < 12.8°C 或 温度 > 37°C** → 停止生长

## 湿度要求

### 最适空气相对湿度
- **值**: 75%
- **配置**: optimalHumidity = 75%

### 适宜空气相对湿度范围
- **范围**: 65%～80%
- **配置**: humidityTolerance = 7.5%
- **计算**: 75 ± 7.5 = 67.5～82.5%（接近实际范围）

### 生长逻辑
- ✅ **65% ≤ 湿度 ≤ 80%** → 可以生长
- ❌ **湿度 < 65% 或 湿度 > 80%** → 停止生长

## 综合生长条件

### 继续生长
必须**同时满足**以下条件：
1. 温度在 12.8～37℃ 范围内
2. 湿度在 65%～80% 范围内

### 停止生长
只要**任意一个**条件不满足：
1. 温度 < 12.8℃ 或 温度 > 37℃
2. 湿度 < 65% 或 湿度 > 80%

## 默认初始值

### 游戏开始时
- **温度**: 26°C（最适温度）
- **湿度**: 75%（最适湿度）
- **状态**: ✅ 完美生长条件

### 动态变化
- **温度变化速度**: 1°C/秒
- **湿度变化速度**: 2.5%/秒
- **变化范围**: 
  - 温度: -20°C ～ 50°C
  - 湿度: 0% ～ 100%

## 游戏难度分析

### 温度安全区间
- **适宜范围**: 12.8～37℃（宽度24.2℃）
- **初始值**: 26℃
- **距离下限**: 26 - 12.8 = 13.2℃（13.2秒后到达）
- **距离上限**: 37 - 26 = 11℃（11秒后到达）

### 湿度安全区间
- **适宜范围**: 65%～80%（宽度15%）
- **初始值**: 75%
- **距离下限**: 75 - 65 = 10%（4秒后到达）
- **距离上限**: 80 - 75 = 5%（2秒后到达）

### 难度评估
- **湿度**: ⚠️ 较难（范围窄，初始值偏上限，2秒就可能超出）
- **温度**: ✅ 较易（范围宽，初始值居中，11秒后才可能超出）
- **综合**: 中等难度，需要玩家持续关注湿度

## 示例场景

```

```

```
结果: 停止生长 ⏸️
```

### 场景4：边界值
```
温度: 37°C ✅ (刚好在上限)
湿度: 80% ✅ (刚好在上限)
结果: 继续生长 🌱
```
