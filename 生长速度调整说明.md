# 橘子树生长速度调整说明

## 当前配置

**生长速率**: 3.33%/秒  
**完整生长时间**: 约 30 秒（在最佳环境条件下）

## 计算公式

```
完整生长时间 = 100% ÷ baseGrowthRate ÷ 环境因子
```

在最佳环境条件下（温度25°C，湿度60%，光照600lux）：
- 环境因子 = 1.0（100%）
- 完整生长时间 = 100 ÷ 3.33 ÷ 1.0 ≈ 30 秒

| 阶段 | 生长阈值 | 到达时间 | 持续时间 |
|------|---------|---------|---------|
| 种子 (Seed) | 0% | 0秒 | 3秒 |

## 环境影响

实际生长时间会受环境条件影响：

### 最佳条件（环境因子 = 1.0）
- 温度: 25°C
- 湿度: 60%
- 光照: 600 lux
- **生长时间**: 30 秒

### 良好条件（环境因子 ≈ 0.8）
- 温度: 20°C 或 30°C
- 湿度: 50% 或 70%
- **生长时间**: 约 37.5 秒

### 一般条件（环境因子 ≈ 0.5）
- 温度: 17°C 或 33°C
- 湿度: 45% 或 75%
- **生长时间**: 约 60 秒

### 恶劣条件（环境因子 < 0.3）
- 温度: < 15°C 或 > 35°C
- 湿度: < 40% 或 > 80%
- **结果**: 树木死亡，停止生长

## 如何调整生长速度

### 方法一：修改配置文件（推荐）

1. 在 Unity 中找到配置文件：
   - 路径: `Assets/Scripts/OrangeTree/OrangeTreeConfig.asset`
2. 在 Inspector 中修改 `Base Growth Rate` 值
3. 保存场景

### 方法二：直接修改资源文件

编辑 `TreePlanQAQ/Assets/Scripts/OrangeTree/OrangeTreeConfig.asset`：

```yaml
baseGrowthRate: 3.33  # 修改这个值
```

## 注意事项

1. **环境影响**: 实际生长时间会受环境条件影响，上述时间是在最佳条件下的理论值
2. **死亡状态**: 如果环境条件恶劣（温度或湿度超出范围），树木会停止生长
3. **Range 限制**: `baseGrowthRate` 的范围限制在 0.1 - 10.0 之间
4. **性能考虑**: 过快的生长速度可能导致阶段切换过于频繁

## 测试建议

1. 运行游戏
2. 确保环境参数在最佳范围内：
   - 温度: 25°C
   - 湿度: 60%
3. 点击"开始生长"
4. 使用秒表计时，验证是否在30秒内完成
